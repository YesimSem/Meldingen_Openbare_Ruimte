# Meldingen_Openbare_Ruimte
Ik heb de open data van Gemeente Den Haag – Meldingen Openbare Ruimte geëxploreerd. In dit portfolio staan interactieve barplots en een bubble map, gemaakt met Plotly. 

Als je de interactieve figuren wilt bekijken, vind je hier de bijbehorende links:
https://nbviewer.org/github/YesimSem/Meldingen_Openbare_Ruimte/blob/main/bubblemap.html

Databron: https://denhaag.dataplatform.nl/#/data/1ffa4758-d143-4d78-b510-a713ab8365df
